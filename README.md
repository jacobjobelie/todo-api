# React Redux example.

This is the code for the tutorial you can find [here](https://dev.to/svinci/react-js-web-site-example-almost-real-life-like).

To run this project you'll need NodeJS 6 and Python 2.7.

* On one console

```
cd todo-api
npm install
npm run start
```

* On a second console

```
cd todo-site
npm install
npm run serve
```
